## 🛠️ Tech Stack

- React + Vite  
- Tailwind CSS  
- GSAP (GreenSock Animation Platform)  
- Lenis (Smooth Scrolling)

---

## ✨ Features

- Smooth scrolling experience  
- Advanced animations using GSAP  
- Clean and modern UI design  
- Fully responsive layout  
- High-performance frontend
